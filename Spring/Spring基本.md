# Spring基本

## Spring的主要功能：

1. 帮助我们管理对象之间的依赖关系及其生命周期；
2. 提供像通用日志记录、性能统计、安全控制、异常处理等面向切面的能力；
3. 帮助我们管理数据库事务，本身提供了一套简单的JDBC访问实现，并且易于与第三方数据访问框架集成；
4. 提供一套自己的web层框架Spring MVC；
5. 提供粘合其他技术和框架的能力，易于与其他的三方的库整合。
